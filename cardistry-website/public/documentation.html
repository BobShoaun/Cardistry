Getting started with Cardistry

Installation

There are several ways to install this library into your project.

Using NPM:

npm install cardistry

Using Yarn:

yarn add cardistry

Using a CDN (Add this to the head of your HTML document)

<script defer src="unpkg.com/cardistry@3.0.0/:file"></script> <script defer src="https://cdn.jsdelivr.net/npm/cardistry@3.0.0"></script>
